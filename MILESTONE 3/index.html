Execute backend analysis scripts and inspect their JSON output

🧠 Sentiment Analysis
Runs VADER (or rule-based fallback) on all post text, classifies each post as positive, neutral, or negative, and writes scores to sentiment_results. 

        


        
        

          
📊 Data Aggregation

          

            Computes monthly engagement totals per user — likes, shares, comments, views —
            and writes summaries to aggregation_reports.
          

          
          

        


        
        

          
📈 Trend Analysis

          

            Identifies trending hashtags, calculates engagement velocity (engagement/day),
            and compares platform performance. Read-only — no DB writes.
          

          
          

        


      


      

        
HOW IT WORKS

        

          Clicking Run sends a POST request to php/run_script.php,
          which whitelists the script name, constructs a safe shell_exec() call
          to python3, and returns the captured JSON output.
          Scripts read from and write back to MySQL using mysql-connector-python.
          When the DB is unavailable, each script automatically falls back to embedded demo data.
